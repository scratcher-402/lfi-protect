# Пример уязвмиого приложения (сервис заметок)

## Возможности
- Регистрация и вход по логину и паролю
- Создание, редактирование и удаление заметок
- Прикрепление файлов к заметкам
- Выдача доступа к заметкам другим пользователям

## Уязвимости
1. **LFI** в эндпоинте `/download` - позволяет скачать файл по произвольному пути. Для демонстрации защиты от утечек принимаются разные форматы: данные формы, JSON-запрос, а также JSON-запрос, закодированный в base64.
2. **File Upload** в функции сохранения файла. Позволяет записывать и перезаписывать фалы по произвольному пути, если права на запись есть у веб-приложения. Можно раскрутить до **RCE** путем перезаписи исходного кода (`app.py`), который автоматически перезапустится, потому что включен режим отладки Flask.

## Зависимости
1. **Python** 3.10 и выше
2. Библиотеки для Python - см. `requirements.txt`

## Развёртывание:
Установите зависимости:
```
pip install -r requirements.txt
```
Запустите приложение:
```
python3 app.py
```
